Implementasi Teori Graf pada Kontrol Rubik 3D Berbasis Gestur Tangan

Requirements
Sebelum menjalankan aplikasi ini, pastikan sudah terinstall:
1. Python (Versi 3.9 - 3.13)
2. Node.Js (Versi 18+ disarankan)
3. Webcam/Kamera aktif

Instalasi
1. Ekstrak folder project ini ke dalam komputer.
2. Run file `install.bat`.
3. Tunggu hingga proses instalasi modul Python dan Node.js selesai (akan muncul tulisan INSTALASI SELESAI).

Run Apps
1. Run file `start_app.bat`.
2. Script ini akan otomatis menyalakan server AI Kamera, server Web, dan membuka browser Anda di alamat `http://localhost:5173`.
3. Berikan izin penggunaan kamera pada browser Anda (jika diminta).
